```
现在做一个图，内容是用户输入的原图和模型输出ai图的对比图，根据图片的比例和大小采用合适的排列形式，比如上下排列、左右排列或者其他更加有趣的排列方式，内容基于之前的对话内容。不需要文字注释。尽量让两个拼图占满空间
```
